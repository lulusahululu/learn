An **Intersection Type** combines multiple types into **one**. The resulting type must satisfy **all** of the types.

#### âœ… Syntax:
```ts

type B = { age: number };

```

Now `Person` must have both `name` and `age`.

#### ðŸ” Example:
```ts

  name: "Alice",
  age: 30,

```

> Useful when you want to **merge multiple type definitions**.
